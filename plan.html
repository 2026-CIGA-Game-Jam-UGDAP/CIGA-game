Zero-G 漂浮系统

PlayerController.cs

把横版跳跃改成太空惯性漂浮。核心思路：关掉重力、 关掉跳跃、加线性阻尼让玩家自然减速漂浮。

改动清单

  • Rigidbody2D.gravityScale → 0（无重力）
  • 删除 isGrounded / groundCheck 相关代码
  • 删除跳跃逻辑（IsJumpPressed / jumpForce）
  • linearDrag 降低到 0.3~0.5，产生"惯性漂移"手感
  • 保留 moveForce / maxSpeed（后续 jetpack 用）
  • 删除水平移动的 FixedUpdate 加力逻辑（移到 jetpack 模块）
gravityScale=1 jumpForce / 跳跃 groundCheck / isGrounded gravityScale=0 linearDrag=0.3 太空漂移 保留 Rigidbody2D 物理

关键代码差异

// Awake() — 原来是 linearDrag，现在额外关重力 void Awake() { rb = GetComponent<Rigidbody2D>(); // ★ 太空：无重力 + 低阻尼漂浮 rb.gravityScale = 0f; rb.drag = 0.3f; // 越低越滑 } // Update() — 删除地面检测和跳跃，只留输入 void Update() { // 输入处理（jetpack 模块接管） ReadJetInput(); } // FixedUpdate() — 删除旧的横向加力，等 jetpack 重写 void FixedUpdate() { // 由 JetpackThrust() 处理（Module 2 加入） }

喷气背包 · 四方向推进

PlayerController.cs

WASD 每个方向独立喷气。按住键 = 持续在那个方向加力，松开 = 惯性漂移。 两个玩家各自独立控制。

输入映射

方向P1 (WASD)P2 (方向键)
↑ 上W↑ Arrow
↓ 下S↓ Arrow
← 左A← Arrow
→ 右D→ Arrow
// FixedUpdate — 四方向喷气推力 void FixedUpdate() { Vector2 input = GetJetInput(); // WASD → Vector2 if (input.sqrMagnitude > 0.01f) { // 有能量才能喷（Module 3 接入） rb.AddForce(input.normalized * jetForce, ForceMode2D.Force); ConsumeEnergy(input.magnitude); // Module 3 } // 惯性限速（所有方向统一上限） rb.velocity = Vector2.ClampMagnitude(rb.velocity, maxSpeed); } // 只读输入，不处理物理 Vector2 GetJetInput() { Vector2 v = Vector2.zero; if (playerIndex == 0) { if (Input.GetKey(KeyCode.W)) v.y += 1; if (Input.GetKey(KeyCode.S)) v.y -= 1; if (Input.GetKey(KeyCode.A)) v.x -= 1; if (Input.GetKey(KeyCode.D)) v.x += 1; } else { if (Input.GetKey(KeyCode.UpArrow)) v.y += 1; if (Input.GetKey(KeyCode.DownArrow)) v.y -= 1; if (Input.GetKey(KeyCode.LeftArrow)) v.x -= 1; if (Input.GetKey(KeyCode.RightArrow)) v.x += 1; } return v; }
jetForce=6~8 全方向加力 惯性限速 Vector2.ClampMagnitude 旧的只有X轴加力

喷气背包 · 能量系统

PlayerController.cs

喷气消耗能量，停止喷气自动回复。能量归零不能喷气。Inspector 里调数值找手感。

参数设计

maxEnergy=100能量上限
drainRate=25/s每秒消耗
regenRate=12/s每秒回复
regenDelay=1s停止喷气后延迟

能量条示意（占位，实际用 UI Image fill）

// 能量变量 [SerializeField] float maxEnergy = 100f; [SerializeField] float drainRate = 25f; [SerializeField] float regenRate = 12f; [SerializeField] float regenDelay = 1f; float currentEnergy; float lastJetTime; // Time.time 上次喷气时间 void Start() { currentEnergy = maxEnergy; } void Update() { // 喷气后延迟 regenDelay 秒才开始回复 if (Time.time - lastJetTime > regenDelay) currentEnergy = Mathf.Min(maxEnergy, currentEnergy + regenRate * Time.deltaTime); } bool TryConsumeEnergy(float amount) { if (currentEnergy <= 0f) return false; currentEnergy -= amount; if (currentEnergy < 0f) currentEnergy = 0f; lastJetTime = Time.time; return true; }

锚点吸附系统

AnchorPoint.cs（新建） + PlayerController.cs

场景中放置锚点。玩家靠近锚点时按 吸附键（如 F / 右Shift）， 快速飞到锚点位置。可用于太空探索中的"抓握跳跃"。

① 靠近锚点锚点半径内
② 提示弹出按F吸附
③ 按下按键触发吸附
④ 飞到锚点速度清零

AnchorPoint 脚本（新建）

// AnchorPoint.cs — 挂在场景中的锚点 GameObject 上 public class AnchorPoint : MonoBehaviour { [SerializeField] float snapRadius = 3f; // 玩家多远能吸附 [SerializeField] float snapSpeed = 12f; // 吸附飞行速度 [SerializeField] SpriteRenderer indicator; // 可选：提示图标 bool playerInRange; void OnTriggerEnter2D(Collider2D other) { if (other.GetComponent<PlayerController>()) playerInRange = true; } void OnTriggerExit2D(Collider2D other) { if (other.GetComponent<PlayerController>()) playerInRange = false; } void Update() { if (playerInRange && Input.GetKeyDown(KeyCode.F)) { // 找到范围内的玩家，触发吸附 var player = FindNearbyPlayer(); if (player != null) player.StartSnapTo(transform.position, snapSpeed); } } }

PlayerController 吸附方法

// 加到 PlayerController public void StartSnapTo(Vector3 target, float speed) { StartCoroutine(SnapRoutine(target, speed)); } IEnumerator SnapRoutine(Vector3 target, float speed) { isSnapping = true; rb.velocity = Vector2.zero; while (Vector3.Distance(transform.position, target) > 0.1f) { transform.position = Vector3.MoveTowards( transform.position, target, speed * Time.deltaTime); yield return null; } transform.position = target; rb.velocity = Vector2.zero; isSnapping = false; }
