# 📄 CIGA 策划导航


## 🎯 核心设计原则

任何策划决策都必须通过这四根柱子的检验：

| 原则 | 核心问题 |
|------|---------|
| 🟢 **低认知压力** | 新人看一局就能上手吗？ |
| 🔵 **可迭代** | MVP 能在 4 小时内可玩吗？ |
| 🟡 **好玩** | 去掉美术音效，核心循环本身有趣吗？ |
| 🟣 **循环性** | 连续玩 5 局，体验有明显不同吗？ |

> 详见：[四柱检验参考](references/four-pillars.md)

---

## 关联技能

> Claude Code 中调用 `/game-design-coach`，输入你的游戏想法，自动产出 GDD + 设计评审 + 范围分析 + 扩展点清单。

| 技能 | 关系 | 说明 |
|------|------|------|
| **grill-me** | 🔗 内部调用 | game-design-coach 在 Step 1-2 中会自动调用来压测关键设计决策 |
| **writing-plans** | 🔗 可选桥接 | Step 6 提示你如需 TDD 实现计划可调用 `/writing-plans`，不强制 |
| **skill-creator** | 🔧 修改工具 | 想改需求或调技能行为时使用（见下方说明） |

## 主题公布后要改什么

Game Jam 公布主题后，打开 `references/jam-card-game-patterns.md`，全文搜索 **三国杀** → 替换为你的对标游戏。

```
示例：三国杀 → 炉石传说
     杀/闪/桃 → 攻击/防御/回复
     武将 → 英雄
```

其他文件通常不需要改，直接开用。

---

## 想改需求或技能行为？

用 **skill-creator**：

```
/skill-creator 我想修改 game-design-coach，
让它每次输出后强制问我是否要生成实现计划
```

或者针对本次 jam 主题做调整：

```
/skill-creator 这次 jam 主题是 "Growth"，
帮我调整 game-design-coach，让所有设计决策都必须围绕成长感展开
```

---

## 快速开始

```
/game-design-coach 海盗主题 2 人卡牌对战，48h jam
```

Skill 会自动完成：
1. 提炼核心定位 → 2. 填充 GDD 模板 → 3. 四柱检验 → 4. 范围分析 → 5. 输出 4 个文件
